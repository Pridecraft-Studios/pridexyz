# <center>{pack_friendly_name}</center>

![Banner showing the modified Hearts (Blinking, Half, Hardcore, etc).]({upload_gallery_url})

Replaces the default Minecraft Hearts with a **{pack_friendly_name}**. 

Includes hardcore, blinking, and half heart variants. Does not change heart types applied by status effects or the environment (Wither, Absorption, Freezing, etc.).

Looking for different colors/flags or styles? Check out the [Pride Hearts Organization Page](https://modrinth.com/organization/pride-hearts)!
